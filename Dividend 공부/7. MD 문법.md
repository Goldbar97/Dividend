# MD 문법
`MD` 는 `MarkDown` 을 의미하며 문서를 작성하는 마크업 언어의 일종이다.

## 제목
```
# title
## title
### title
### title

title
===

title
---
```

## 리스트
```
* list
    * sublist
- list
    - sublist
1. list
    1. sublist
```

* list
    * sublist
- list
    - sublist
1. list
    1. sublist

## 강조
```
**text**
__text__

*text*
_text_
```

**text**
__text__

*text*
_text_

## 링크
```
[구글](https://www.google.com)
```

[구글](https://www.google.com)
